Sobre Este Projeto

Este repositório serve como um ambiente prático de estudo do meu curso de desenvolvimento de jogos 3D. O objetivo é aplicar diretamente os conceitos teóricos da Unity e C# em um projeto funcional.

ATENÇÃO: O código aqui pode estar em constante mudança e pode conter soluções experimentais, refletindo o processo de aprendizado.

Funcionalidades Implementadas (Focus do Módulo)

Os principais sistemas e mecânicas que estou implementando e testando nesta fase são:

Movimentação: Sistema de movimento 3D com `CharacterController`.
Controle de Input: Utilização do *New Input System* da Unity.
Patrulha e IA: Navegação básica com `NavMeshAgent` (NPC_IA).
Animação: Controle de estados (`Move`, `Jump`, `Attack`) via `Animator`.
Detecção de Colisão: Utilização de Triggers (`OnTriggerEnter`) para detectar interação.

Requisitos para Rodar

Para abrir e rodar este projeto, você precisará ter:

1.  Unity Engine: Unity 6.2.
2.  Visual Studio ou IDE compatível.

Como Iniciar

1.  Clone o repositório.
2.  Abra a pasta clonada através do **Unity Hub**.
3.  Navegue até `Assets/Scenes` e abra a cena principal (ex: `SampleScene`).
